# vue3-learning

## 搭建项目
- 安装最新版本 @vue/cli@next
- 创建脚手架工程
- 安装 ant-design 和 axios
- 启动项目

```shell
yarn global add @vue/cli@next
# OR
npm install -g @vue/cli@next

vue create vue3-learning

cd vue3-learning
yarn add ant-design-vue@next axios
yarn serve
```













## Project setup

npm install
